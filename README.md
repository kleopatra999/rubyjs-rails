Rails asset-pipeline gem to provide [RubyJS](http://rubyjs.org/)

# Install

  gem 'rubyjs-rails'


# Use

In your application.js manifest:

  //= require ruby

or directly available under `/assets/ruby.js`

Easy as pie.
